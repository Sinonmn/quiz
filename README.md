### Prerequisites

- [Node.js](https://nodejs.org/) (v18 or higher recommended)
- npm or yarn

### Installation

1.  **Clone the repository:**
    ```bash
    git clone [https://github.com/sinonmn/quiz.git](https://github.com/sinonmn/quiz.git)
    ```
2.  **Navigate to the project folder:**
    ```bash
    cd quiz
    ```
3.  **Install dependencies:**
    ```bash
    npm install
    ```
4.  **Environment Setup:**
    Create a `.env` file and add your backend URL:
    ```text
    VITE_API_URL=[https://your-api-on-koyeb.koyeb.app](https://your-api-on-koyeb.koyeb.app)
    ```
5.  **Launch the development server:**
    ```bash
    npm run dev
    ```
